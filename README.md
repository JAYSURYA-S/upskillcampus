# LinkShrink - URL Shortener

LinkShrink is a clean, modern URL shortener built with Python and Flask. It allows users to convert long, cumbersome URLs into short, manageable links.

## Features
- **Modern UI:** Built with Material Design principles and Bootstrap 5.
- **Fast Shortening:** Uses a randomized 6-character unique ID system.
- **Local Database:** Uses SQLite for lightweight, persistent storage.
- **One-Click Copy:** Integrated clipboard functionality.
- **Mobile Responsive:** Works perfectly on phones and tablets.

## Quick Start
1. Ensure Python is installed.
2. Run the application:
   ```powershell
   cd url_shortener
   .\run.ps1
   ```
3. Visit `http://127.0.0.1:5000` in your browser.

## Technical Details
- **Backend:** Flask (Python)
- **Database:** SQLite
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5
